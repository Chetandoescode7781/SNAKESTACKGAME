# 🐍 Web-Based Snake Game

A simple Snake game built using HTML5 Canvas and JavaScript. This is a browser-based version of the classic Snake game with a few extra features added for better gameplay.

---

## 🎮 Features

- Classic Snake gameplay  
- Different food types:
  - Normal Food (+1 score)
  - Bonus Food (+5 score)
  - Immune Food (temporary wall pass ability)  
- Difficulty selection (changes speed)  
- Pause/Resume functionality  
- Backend integration for saving scores  

---

## 🕹️ Controls

- Arrow Keys / WASD → Move the snake  
- Spacebar → Pause / Resume  

---

## ⚙️ Technologies Used

- HTML5 Canvas  
- JavaScript  
- CSS  
- Python (Backend for score storage)  

---
